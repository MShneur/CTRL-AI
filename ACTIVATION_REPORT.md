## 2. What Do You Actually Have? (Plain English)

You have a **behavior upgrade kit** for any AI you already use. The main file — `CTRL-AI.md` — is like a very detailed job description you hand to an employee on their first day. Once the AI reads it, it stops acting like an eager-to-please assistant and starts acting like a disciplined, skeptical expert committee.

The framework currently ships at **Version 5.2.1**, which is the most stable release. The big features in this version:

* **The Internal Judge:** Every answer the AI gives gets secretly graded 0–10 before you see it. If the grade is below 8, the AI rewrites the answer silently on its own.
* **Escape Hatch:** If a chat session gets too long and the AI starts forgetting things, type `CTRL_MIGRATE` and the AI packages up your entire project into a portable "save file" you can drop into a fresh chat.
* **Anti-Yes-Man Alarm:** If the AI agrees with you 3 times in a row without pushback, it automatically triggers a Devil's Advocate check on itself. It will now log this in every message so you can see it happening.
* **Evidence Labels:** Every fact the AI states is labeled as `[EVIDENCE]` (verified), `[PRACTICE]` (common standard), or `[SPECULATIVE]` (a guess). You always know what you can trust.

## 3. How Do You Install It? (Two Easy Methods)

### Method A — The Quick Install (5 minutes, for everyday use)

1. Open `CTRL-AI.md` and scroll to **Section 14**.
2. Find the block for your platform (ChatGPT, Claude, DeepSeek, or Gemini).
3. Copy the entire text block.
4. Paste it into your AI's **"Custom Instructions"** or **"System Prompt"** settings.
5. Done. Every new chat you start is now running CTRL-AI.

### Method B — The Deep Dive (for big projects)

1. Open a brand-new, empty chat.
2. Copy the **entire contents** of `CTRL-AI.md`.
3. Paste it as your very first message.
4. The AI will confirm it has loaded the framework and ask you what project you want to start.
