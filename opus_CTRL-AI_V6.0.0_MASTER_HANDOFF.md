## 11. OPEN RESEARCH QUESTIONS

1. **Is CTRL-AI's structure optimal or just interpretable?** Literature validates structure matters, not that THIS structure is best. Ablation testing needed.
2. **Does SCEL actually prevent sycophancy or just add a flag?** Does the model produce genuinely different output after trigger?
3. **What is the real token overhead of governance?** No empirical data comparing governed vs ungoverned sessions on identical tasks.
4. **Can partial adoption work?** Docs say "partial is not CTRL-AI." Is there a minimum viable subset?
5. **How does governance degrade across model generations?** If reasoning-native models ignore scaffolding, does Behavior still hold?
6. **What is the failure mode distribution?** When governance drifts, what breaks first and why?
7. **Is the Lexical Matrix token-positive?** Does verb expansion save more downstream than it costs upfront?
